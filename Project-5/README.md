# Project 5: Predicting House Prices with Linear Regression

## Overview
This project focuses on building a Linear Regression model to predict house prices based on various property features. The goal is to develop a predictive model that estimates house prices accurately and provides insights into the factors influencing property values.

## Objectives
- Analyze housing market data.
- Perform data cleaning and preprocessing.
- Select relevant features for prediction.
- Build and train a Linear Regression model.
- Evaluate model performance using appropriate metrics.
- Visualize relationships between actual and predicted prices.

- Mean Absolute Error (MAE)
