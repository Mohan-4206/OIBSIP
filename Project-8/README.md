# Project 8: Google Play Store Data Analysis

## Overview
This project analyzes the Google Play Store dataset to uncover insights into the Android app market. The analysis involves cleaning, merging, and exploring two datasets—app details and user reviews—to understand app performance, user sentiment, category distribution, pricing trends, and market dynamics.

## Objectives
- Clean and preprocess Google Play Store datasets.
- Merge app information with user review data.
- Analyze app categories and market distribution.
- Examine ratings, installs, size, pricing, and popularity.
- Perform sentiment analysis on user reviews.
- Create visualizations to present meaningful business insights.
