# Project 1: Retail Sales Analysis

## Overview
This project focuses on Exploratory Data Analysis (EDA) of a Retail Sales Dataset to identify sales trends, customer purchasing behavior, and business insights. The analysis was performed using Python in Google Colab.

## Objectives
- Analyze retail sales data.
- Identify customer and product trends.
- Generate business insights through data visualization.
- Understand revenue patterns and sales performance.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Tasks Performed
- Data Loading and Inspection
- Data Cleaning
- Statistical Analysis
- Sales Trend Analysis
- Customer Behavior Analysis
- Data Visualization

## Files
- `EDA_Retail_Sales_Analysis.ipynb`

## Outcome
Successfully extracted meaningful insights from retail sales data and visualized important business trends to support data-driven decision-making.
