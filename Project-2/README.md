# Project 2: Customer Segmentation Analysis

## Overview
This project focuses on Customer Segmentation Analysis for an e-commerce business. The objective is to analyze customer behavior and purchasing patterns to group customers into distinct segments. These segments can help businesses design targeted marketing campaigns, improve customer satisfaction, and optimize business strategies.

## Objectives
- Analyze customer purchase behavior and transaction patterns.
- Perform data cleaning and preprocessing.
- Calculate key customer metrics.
- Segment customers into meaningful groups using clustering techniques.
- Visualize customer segments and purchasing trends.
- Generate actionable business insights and recommendations.
